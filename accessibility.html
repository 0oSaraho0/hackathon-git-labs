Accessibility Policy

We believe that everyone should benefit from this website and get the most out of it.

Supported Browsers

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge
  • Samsung Internet

How to increase text size

  • Google Chrome
    • Using a mouse: control key and roll forward mouse wheel forward or backwards to preferred text size.
    • Using a keyboard: control key and + (plus) or - (minus) to increase or decrease text size.
  • Firefox
    • Using a mouse: control key and roll forward mouse wheel forward or backwards to preferred text size.
    • Using a keyboard: control key and + (plus) or - (minus) to increase or decrease text size.
  • Safari
    • Using a mouse: control key and roll forward mouse wheel forward or backwards to preferred text size.
    • Using a keyboard: control key and + (plus) or - (minus) to increase or decrease text size.
    • Using a keyboard: Command key and + (plus) or - (minus) to increase or decrease text size (Mac users)
  • Edge
    • Using a mouse: control key and roll forward mouse wheel forward or backwards to preferred text size.
    • Using a keyboard: Command key and + (plus) or - (minus) to increase or decrease text size
    • Using a keyboard: Command key and + (plus) or - (minus) to increase or decrease text size (Mac users)

Screen resolution

This website has been designed to be fully responsive, from the minimum width screen sizes 320px to the maximum width screen size of 2000px.
